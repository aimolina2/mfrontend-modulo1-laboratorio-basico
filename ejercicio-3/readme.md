# Laboratorio Layout BASICO. Ejercicio 3.

En la barra de navegación encontramos los ítems alineados usando flex y distribuyéndolos con space-around.

A partir de una resolución de 768px el menún pasa a mostrarse de forma vertical colocándose un ítem debajo del otro.

Para ello cambiamos la dirección de flex:

```
flex-direction: column;
```
